# 🤖 SmartChat AI

SmartChat AI is a full-stack AI chatbot built using Flask, JavaScript, and the Groq API. It supports real-time conversations, browser-based voice input, and speech output for an interactive chat experience.

---

## 🚀 Features

- 💬 Real-time AI chat
- 🎤 Voice input using browser (Speech Recognition API)
- 🔊 Voice output (Text-to-Speech)
- ⚡ Fast responses using Groq API
- 🧠 Chat history stored using SQLite
- 🎨 Simple and clean UI

---

## 🛠️ Tech Stack

- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript
- AI API: Groq (LLaMA models)
- Database: SQLite
